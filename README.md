# CSC3002-CppBasic-PersonalNotes
该仓库为针对 香港中文大学（深圳） CSC3002（C++基础） 课程整理的个人学习笔记。
笔记内容基于个人学习与理解整理，力求完整但不保证覆盖课程全部细节，适合用于课前预习时建立整体认知，或在复习阶段回顾课程的核心框架与主要概念。

This repository contains personal study notes for **CSC3002 (C++ Fundamentals)** at The Chinese University of Hong Kong, Shenzhen.
The notes are compiled based on personal learning and understanding. While an effort has been made to maintain completeness, the content does not aim to cover every detail of the course. They are intended to provide a high-level overview for pre-study and to serve as a structured reference when reviewing the core concepts and overall framework of the course.
